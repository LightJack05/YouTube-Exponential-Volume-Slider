# YouTube Exponential Volume Slider
Makes the YouTube video volume slider exponential so it's easier to select lower volumes.
Humans perceive loudness logarithmically, so exponential scaling makes the slider feel more natural and precise at low volumes.

Install [Tampermonkey](https://www.tampermonkey.net/) for your browser. Then add a new userscript and paste the code from `youtube_exponential_volume_slider.js`
